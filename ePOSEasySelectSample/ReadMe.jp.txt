1.本ソフトウェアについて
  ePOS-Print SDK for Android ePOSEasySelectSample は、
  EasySelect ライブラリを使用するためのサンプルプログラムです。
  API仕様については、ePOS-Print SDK for Android ユーザーズマニュアル を参照ください。

2.サンプルプログラムの構成
  libs
    ePOS-Print SDK for Androidとzxingのライブラリです。

  src
    eposeasyselectsample
      quickpairing
        簡単選択のサンプルです。
        camera
          カメラのサンプルです。
        nfc
          NFCのサンプルです。
      printqrcode
        QRコード印刷のサンプルです。


3.zxingについて
  本サンプルプログラムは、バーコードの解析にzxingを利用しています。
  zxingは、apatch2ライセンスにより提供されています。

  zxingについては、以下を参照してください。
    https://github.com/zxing/zxing
